If a wildcard ('*') resolves to a version in the source pattern and would resolve multiple versions only the highest version would get downloaded. E.g assume two artifacts:
path_to_version/2.0/dir/a.ext
path_to_version/2.1/dir/a.ext
and a resolve pattern of
path_to_version/*/dir/**
then only the 2.1 version would get downloaded.
Note: only the first wildcard is tested against a version
